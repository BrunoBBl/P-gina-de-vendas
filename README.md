# Tech Shop - Landing Page de Vendas

Uma página de vendas moderna, clean e responsiva focada em smartwatches, acessórios eletrônicos e gadgets. Ideal para e-commerce, dropshipping ou campanhas promocionais de produtos tech.

## Visão Geral

- Hero section impactante com imagem de smartwatch e call-to-action principal  
- Seção **Best Sellers** destacando 3 produtos em destaque  
- Formulário de **Newsletter** para capturar leads  
- Footer completo com categorias, sobre, redes sociais e suporte ao cliente  
- Design responsivo com cores modernas (azul principal + verde em detalhes)  
- Estilo minimalista e foco em conversão

## Tecnologias Utilizadas

- HTML5  
- CSS3 (Flexbox + posicionamento absoluto para hero)  
- Google Fonts: Heebo (300, 400, 700)  
- Sem frameworks (vanilla HTML/CSS)

## Funcionalidades Principais

- Header fixo com logo customizado e menu de navegação  
- Hero com overlay de texto branco e botão CTA  
- Cards de produtos com tag "New", preço promocional e botão "Add to Bag"  
- Newsletter com input arredondado e botão hover  
- Footer em dark mode com 4 colunas de links  
- Efeitos hover em botões e links
